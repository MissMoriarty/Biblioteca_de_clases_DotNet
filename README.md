# Biblioteca de clases .NET - Fundamentos de POO

El proyecto consiste en una aplicación de consola que documenta, mediante código ejecutable y comentarios detallados, qué son las clases, objetos y métodos, y cómo interactúan entre sí.

## Conceptos y temas abordados

* **Fundamentos:**
* Definiciones de **Clase** (el plano) vs. **Objeto** (la instancia).
* Estructura básica: Estado (propiedades) y Comportamiento (métodos).


* **Constructores:**
* Inicialización de objetos.
* Constructores por defecto vs. personalizados.


* **Métodos:**
* **Estáticos vs. De Instancia:** Diferencias entre métodos que pertenecen a la clase (`static`) y los que requieren un objeto.
* **Tipos de Retorno:** Métodos `void` vs. métodos que devuelven valores (`return`).
* **Parámetros y Argumentos:** Diferencias conceptuales y prácticas.
* **Sobrecarga (Overloading):** Cómo definir múltiples métodos con el mismo nombre pero diferentes firmas (ejemplo de procesador de pagos).


* **Modificadores de Acceso:**
* Explicación de `public`, `private` y `protected`.
